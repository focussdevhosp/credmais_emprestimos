# CredMais Emprestimos

Site institucional da CredMais Emprestimos, com paginas dedicadas para:

- Comerciantes
- Micro emprestimos
- Motoristas de aplicativo
- Entregadores e delivery

## Como abrir

Este projeto e um site estatico. Para visualizar localmente, sirva a pasta com qualquer servidor HTTP local e abra `index.html`.

Exemplo:

```bash
npx serve .
```

## Arquivos principais

- `index.html`
- `comerciantes.html`
- `micro-emprestimos.html`
- `motoristas.html`
- `entregadores.html`
- `styles.css`
- `main.js`
- `assets/images/`
